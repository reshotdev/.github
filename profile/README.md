# PaperJSX

Document generation from JSON. Native PPTX, DOCX, and PDF.

- [`@paperjsx/json-to-pptx`](https://npmjs.com/package/@paperjsx/json-to-pptx) — generate PowerPoint files from JSON
- [`@paperjsx/json-to-pptx-lite`](https://github.com/paperjsx/json-to-pptx-lite) — free version (MIT)
- [`@paperjsx/json-to-docx`](https://npmjs.com/package/@paperjsx/json-to-docx) — generate Word documents from JSON
- [`@paperjsx/mcp-server`](https://github.com/paperjsx/mcp-server) — MCP server for AI agents
- [`@paperjsx/templates`](https://github.com/paperjsx/templates) — ready-to-use document templates
